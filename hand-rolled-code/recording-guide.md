# 初めてのプログラミング動画 撮影・アップロードガイド

手持ちの機器だけで完結させる最小構成。

---

## 必要なもの（追加購入なし）

| 用途 | 使うもの |
|---|---|
| 画面録画 | QuickTime Player（Mac 標準） |
| マイク | MacBook 内蔵マイク（まずはこれで十分） |
| 編集 | iMovie（Mac 標準、無料） |
| アップロード | YouTube Studio（ブラウザ） |

---

## Step 1 — 事前準備（録画前）

### デスクトップを整える
- 不要なウィンドウ・タブをすべて閉じる
- 通知をオフにする → システム設定 → 通知 → 「おやすみモード」をオン
- Dock を自動的に隠す設定にする（画面が広く見える）

### VS Code の設定
- フォントサイズを **18〜20pt** に上げる（録画すると小さく見える）
- テーマはコントラストが高いものが見やすい（例: One Dark Pro）
- ファイルツリーを左に表示した状態にしておく

### ターミナルの設定
- フォントサイズを同様に大きくする
- プロンプトをシンプルにする（長いパスが映らないよう）

---

## Step 2 — 画面録画（QuickTime Player）

1. QuickTime Player を起動
2. メニュー → **ファイル → 新規画面収録**
3. 録画ボタン横の `∨` をクリック → マイクを選択（内蔵マイクまたは外部マイク）
4. **「画面の一部を収録」** を選び、VS Code + ターミナルが映る範囲だけ選択する
   - 全画面より範囲指定のほうが、無駄な余白がなく見やすい
5. 録画開始 → スクリプト通りに作業しながら解説
6. メニューバーの停止ボタン（■）で終了 → `.mov` ファイルが保存される

### 録画中のコツ
- 話し始める前に **2〜3秒の無音** を入れる（編集で切りやすい）
- 噛んだり詰まったりしても止めない。後で編集でカットできる
- コードを書きながら同時に話すのは難しいので、**書いてから説明する** 順番にする
- キーボードの音が気になる場合はマイクを少し遠ざける

---

## Step 3 — 編集（iMovie）

### 基本的な編集だけでOK
1. iMovie を起動 → 新規プロジェクト → **ムービー**
2. 録画した `.mov` ファイルをドラッグしてタイムラインに追加
3. やること：
   - 冒頭と末尾の余分な無音部分をカット
   - 噛んだ・長い沈黙の部分をカット（クリップを分割 → 削除）
   - 必要なら **タイトルテキスト** を冒頭に入れる（iMovie の組み込み機能で可）
4. 完成したら → **ファイル → 共有 → ファイル** → 解像度 1080p で書き出し

### やらなくていいこと（最初は）
- BGM（著作権問題が面倒）
- テロップ全文（時間がかかりすぎる）
- サムネイル画像（後から作れる）

---

## Step 4 — YouTube アップロード

1. [youtube.com/upload](https://youtube.com/upload) にアクセス（Google アカウントでログイン）
2. 書き出したファイルをアップロード
3. 必須入力項目：

   | 項目 | 内容 |
   |---|---|
   | タイトル | 例: `Implementing reset() in a PHP Circuit Breaker library` |
   | 説明 | プロジェクトの GitHub URL + 今回やった内容を2〜3行 |
   | 公開設定 | 最初は **限定公開** にして自分で確認してから公開に変えるのがおすすめ |

4. 処理完了後、**限定公開 URL** でスマホなどから視聴して確認する
5. 問題なければ **公開** に変更

---

## 手元映像のピクチャー・イン・ピクチャー（PiP）

タイピング中だけ右下に手元映像を出す方法。**iMovie で後から合成する方法**が追加機材なしで一番シンプル。

### 撮影：iPhone で手元を録画する

1. iPhone をキーボードの真上〜斜め前に固定する
   - 本を積んでその上に立てかける、またはスマホスタンド（100均で入手可）
   - 手全体とキーボードが映る角度に調整する
2. カメラアプリで録画開始（画面録画と同時に録り始める）
3. 画面録画と手元録画を **同時に開始・終了** する
   - 同期ズレを最小にするため、録画開始直後に手を1回パンと叩く（クラップ）
   - 編集時にこの音波形を目印に位置を合わせる

> **Continuity Camera を使う場合（macOS Sonoma 以降）**  
> iPhone を Mac の近くに置くだけで自動的にウェブカメラとして認識される。  
> QuickTime で「新規ムービー収録」→ カメラに iPhone を選択 → 録画すれば手元映像を別ファイルとして同時収録できる。

### 編集：iMovie で PiP を合成する

1. iMovie のタイムラインに画面録画クリップを追加済みの状態で開始
2. 手元映像クリップをタイムラインの **画面録画クリップの上** にドラッグ
3. クリップ選択 → 画面上部の「ビデオオーバーレイ設定」→ **ピクチャ・イン・ピクチャ** を選択
4. プレビュー画面でクリップを **右下にドラッグ**、サイズを小さく調整
5. PiP クリップの **前後をトリミング** してタイピングしている区間だけに絞る
   - タイピングしていない区間（話しているだけの部分）は PiP を消す

### 注意点
- 手元映像は **少し暗め** になりがち。撮影場所の照明を意識する（デスクライトを手元に向けるだけで改善）
- 右下に PiP を置くと、コードの右端が隠れる場合がある。コーディング中は意識的に左寄りにコードを書くか、PiP を左下に変えるのも手

---

## 音質が気になったら（次のステップ）

内蔵マイクで録ってみて、聞き返して問題なければそのままでOK。  
もし反響・ノイズが気になる場合の無料対策：

- **部屋の中で一番こもった場所**（クローゼットの前など）で録音する
- Audacity（無料）でノイズ除去できるが、最初は不要

---

## チェックリスト（録画前）

- [ ] 通知をオフにした
- [ ] VS Code のフォントサイズを上げた
- [ ] スクリプトを手元に印刷 or サブディスプレイに表示した
- [ ] QuickTime でマイクが選択されている
- [ ] 30秒テスト録音して音声・画質を確認した
- [ ] トイレに行った（途中で止めると編集が面倒）
