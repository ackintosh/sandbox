# Hand-Rolled Code

**Theme**: Handwriting source code by hand — a deliberate contrast to the age of AI coding tools. Every line typed by a human.

## Channel Description

Coding by hand, one keystroke at a time.

In an age where AI writes code for you, I believe there's still value in understanding every line you type. This channel is a place where I sit down, open a text editor, and write real software with my own hands — no AI autocomplete, no generated boilerplate.

I mainly work on open-source projects. Videos cover bug fixes, feature implementation, and code reading.
